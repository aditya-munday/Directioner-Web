IMPLEMENTATION NOTES
Chart data: All chart data should be generated from a generateMockData(dateRange) utility that produces realistic-looking data (not perfectly uniform — add noise/variance).

Responsive charts: SVG charts should use viewBox and preserveAspectRatio so they scale naturally. Use ResizeObserver to get container width and recompute bar widths.

Animation performance: Use will-change: transform on animated elements. Prefer transform and opacity over layout-affecting properties. Use framer-motion's layout prop for smoothly animated layout changes.

Credits progress bar color logic:

const getBarColor = (pct: number) => 
  pct > 80 ? '#FFE500' :    // yellow — normal
  pct > 50 ? '#FFE500' :    // still yellow
  pct > 20 ? '#FFA500' :    // orange — caution
  '#FF4444';                // red — critical
Chart tooltip: All chart tooltips should be a dark #141414 card with yellow top border, positioned above the cursor, with JetBrains Mono text. Use a useTooltip hook that tracks mouse position.

All animations must respect prefers-reduced-motion:

const prefersReduced = window.matchMedia('(prefers-reduced-motion: reduce)').matches;
